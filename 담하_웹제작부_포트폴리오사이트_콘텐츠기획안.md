## 1. 소개서 핵심 분석 요약

### 1-1. 회사 정체성
- **회사명**: 주식회사 담하 (DAMHA)
- **부서**: 웹제작부
- **핵심 슬로건**: "웹을 통한 병원의 새로운 성장 파트너"
- **타겟 고객**: 병원/의료기관 (1인 병원 ~ 대형 병원)
- **소재지**: 부산광역시 동래구 연안로 59번길 7, 엘엔케이빌딩 5층

### 1-2. 핵심 가치 제안 (4가지 차별점)
| 차별점 | 키워드 | 설명 |
|--------|--------|------|
| 무에서 유 만들기 | 브랜드 특화 경험 | 빈 화면에서 브랜드의 첫인상 구축 |
| 구조 검증하기 | 안전성 | 기획~개발까지 내구성 강화 |
| 다양하게 설계하기 | 유연한 솔루션 | 최적의 UX를 찾아가는 다양한 시도 |
| 새로움 추구하기 | 혁신성 | 차별화된 사용자 경험 |

### 1-3. 상품 라인업
| 등급 | 가격대 | 타겟 | 구성 |
|------|--------|------|------|
| **LITE** | ~500만원 | 1인 병원, 소형 의원 | Single ver. (원페이지) |
| **BASIC** | 700만원~ | 지역 병원 | Multi ver. (2depth) |
| **STANDARD** | 1000만원~ | 대형 병원 | Multi ver. + 콘텐츠 기획 |
| **별도협의** | 맞춤 | 종합 병원급 | 풀 커스텀 |

### 1-4. 담하 관리 시스템 (CMS) 기능
**기본 제공**: 사이트 설정, 기본 정보 설정, 진료일정 관리, 의료진 관리, 팝업 관리, 퀵메뉴 관리, 메인 슬라이드 관리, 둘러보기, 장비 관리, 상담 관리, 게시판 관리(5개), 통계
**옵션**: 월별 진료일정 관리, 유튜브 관리

### 1-5. 포트폴리오 현황 (총 15개 프로젝트)
- **LITE 등급** (9개): 다대정내과, 우영하척정형외과, 당당한의원 대구수성, 핑이비인후과, 솔리드피부과, 고운빔의원, 권현영산부인과, 신세계한방병원, 김영복정형외과
- **STANDARD 등급** (3개): 창원당당한방병원, 해운대푸른바다병원 피부과, 부산미남병원
- **별도협의** (3개): 큐병원, 거인병원, 연산당당한방병원
- **진료과목 다양성**: 내과, 정형외과, 한의원, 이비인후과, 피부과, 산부인과, 한방병원 등

```

├── 02. Problem (문제 제기)

│
├── 03. About Us (부서 소개)

│
├── 05. Service (상품 안내)
│   ├── 상품 라인업 비교 (LITE / BASIC / STANDARD)
│   └── 패키지 구성 + 유지관리 서비스
│
├── 06. CMS (담하 관리 시스템)
│   ├── 기능 개요 인터랙티브 쇼케이스

│   ├── 빠른 상담 폼

└── 10. Footer
    ├── 회사 정보
    └── SNS / 블로그 링크
```

## 3. 섹션별 콘텐츠 상세 기획

### 섹션 01: Hero (메인 비주얼)

**목적**: 첫인상 — 무엇을 하는 곳인지 3초 안에 전달

**콘텐츠 구성**:
- 메인 카피: "당신의 병원 홈페이지, 새로운 관점에서 만들겠습니다."
- 서브 카피: "기획부터 운영까지, 웹과 관련된 모든 것을 책임지는 올인원 파트너"
- CTA 버튼 2개: `프로젝트 문의하기` / `포트폴리오 보기`
- 배경: 실제 포트폴리오 사이트들의 디바이스 목업 슬라이드 또는 영상

**디자인 노트**: 소개서의 메인 컬러(딥 퍼플/인디고 #3F3D9E 계열) 활용, 담하 로고 좌상단 고정

---

### 섹션 02: Problem (문제 제기)

**목적**: 잠재 고객의 공감 유도 → 니즈 자극

**콘텐츠 구성** (소개서 p.2 기반):

| Problem | 제목 | 핵심 메시지 |
|---------|------|-----------|
| 01 | 낮은 접근성과 낡은 UI | 업데이트 없이 정체된 홈페이지, 브랜드 가치 전달 실패 |
| 02 | 환자 기대와 서비스 간 격차 | 빠른 정보, 간편 예약, 신뢰감 미충족 → 포털 의존 |
| 03 | 운영의 불편함 | 기능이 있어도 관리가 어려우면 방치됨 |

**마무리 카피**: "환자와 병원을 잇는 새로운 기준의 홈페이지가 필요합니다"

**인터랙션**: 스크롤 시 카드가 순차적으로 나타나는 애니메이션

---

### 섹션 03: About Us (부서 소개)

**목적**: 담하 웹제작부의 역량과 원스톱 체계 어필

**콘텐츠 구성** (소개서 p.3 기반):
- 좌측: 텍스트 블록
  - "우리는 홈페이지를 만들기만 하지 않습니다"
  - 원스톱 제작: 기획 → 디자인 → 퍼블리싱 → 개발
  - 병원 특수성: 의료법 준수, 마케팅 관점, SEO 최적화
- 우측: 역량 레이더 차트 (기획, 디자인, 퍼블리싱, 개발, 의료법 고려, 유지관리, 마케팅, 접근성&SEO)

**마무리 카피**: "웹과 관련된 모든 것을 책임지는 올인원 파트너입니다"

---

### 섹션 04: Difference (차별점)

**목적**: 경쟁사 대비 담하만의 강점 각인

**콘텐츠 구성** (소개서 p.4 기반):

4개의 카드로 구성:

| # | 차별점 | 아이콘 | 핵심 가치 | 설명 |
|---|--------|--------|----------|------|
| 1 | 무에서 유 만들기 | 💡 | 브랜드 특화 경험 | 브랜드의 첫인상을 빈 화면에서 구체적 웹 경험으로 변환 |
| 2 | 구조 검증하기 | 🔧 | 안전성 | 모든 구조의 내구성까지 강화 |
| 3 | 다양하게 설계하기 | 🧩 | 유연한 솔루션 | 실패도 더 나은 경험으로 이어지도록 최적의 길 탐색 |
| 4 | 새로움 추구하기 | 🚀 | 혁신성 | 차별화된 UX로 환자와 병원의 접점 발굴 |

**마무리 카피**: "홈페이지 그 이상, 환자 유입 시스템을 만듭니다"

**인터랙션**: 호버 시 카드 뒤집기 효과로 상세 설명 노출

---

### 섹션 05: Service (상품 안내)

**목적**: 명확한 상품 구조 전달 → 적합한 상품 선택 유도

**콘텐츠 구성** (소개서 p.6~7 기반):

**5-1. 상품 비교 테이블**

| 항목 | LITE (~500만) | BASIC (700만~) | STANDARD (1000만~) |
|------|-------------|---------------|-------------------|
| 구성 | 원페이지(Single) | Multi(2depth) | Multi + 콘텐츠 기획 |
| 타겟 | 1인 병원, 소형 의원 | 지역 병원 | 대형 병원 |
| 키워드 | #빠른구축 #경제적 | #아이덴티티 #맞춤형 | #구조설계 #콘텐츠기획 |
| CTA | 견적 문의 | 견적 문의 | 견적 문의 |

**5-2. 공통 서비스**

**패키지 구성**:
1. 정식 오픈 전 임시 사이트 제공
2. 고객 니즈 맞춤 상품 선택
3. 병원 특수성과 전문성 반영 기획
4. 병원 아이덴티티 브랜딩 디자인

**유지관리 서비스**:
1. 6개월 무상 유지관리 (호스팅/도메인/인증서, 기능 하자, 텍스트 수정, 이미지 교체, 스크립트 삽입)
2. 시스템 장애 빠른 대응
3. CMS(담하 관리 시스템) 제공
4. 신속한 소통 환경

### 섹션 06: CMS (담하 관리 시스템 쇼케이스)

**목적**: CMS의 직관성과 편의성을 시각적으로 보여줌 → "이 정도면 나도 관리할 수 있겠다" 인식 유도

**콘텐츠 구성** (소개서 p.9~20 기반):

탭 UI로 주요 기능 12가지를 인터랙티브하게 전시:

| 탭 | 기능명 | 핵심 포인트 |
|----|--------|-----------|
| 1 | 사이트 설정 | 홈페이지 제목, 도메인, 방문자분석 스크립트 |
| 2 | 기본 정보 설정 | 시작페이지, 회사 정보, 네이버 지도 API |
| 3 | 진료일정 관리 | 운영시간 수정, 공통 안내사항, 시간 표시 설정 |
| 4 | 의료진 관리 | 프로필 등록, 주간 일정, 약력, 순서 변경 |
| 5 | 팝업 관리 | 노출 기간, 스타일(슬라이드/탭/컬럼), 링크 연결 |
| 6 | 퀵메뉴 관리 | 전화상담, 카톡, 네이버예약 등 링크 관리 |
| 7 | 메인 슬라이드 | PC/Mobile 이미지, 노출/미노출 |
| 8 | 둘러보기 | 시설 이미지 갤러리, 드래그 순서 변경 |
| 9 | 장비 관리 | 장비명, 설명, 별도 페이지 노출 설정 |
| 10 | 상담 관리 | 빠른상담 DB 확인, 메모, 문자알림 연동 |
| 11 | 게시판 관리 | 공지사항/온라인상담/이벤트/전후사진/치료후기 |
| 12 | 월별 진료일정 | 캘린더 기반 의료진별 일정 관리 (옵션) |

**표현 방식**: 각 탭 클릭 시 실제 관리자 화면 스크린샷(또는 목업) + 우측에 기능 설명 카드

### 섹션 08: Process (진행 프로세스)

**목적**: 협업 과정의 투명성 → 불안감 해소

**프로세스 단계**:
