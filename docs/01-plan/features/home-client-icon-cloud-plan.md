# PortfolioPreview (클라이언트 아이콘 클라우드 & 좌측 타이포그래피 리디자인) 계획안

## 1. 개요
홈 화면의 `PortfolioPreview` 섹션(성공의 증명)을 시니어 디자이너 및 프론트엔드 엔지니어의 관점에서 전면 리뉴얼합니다.
기존 우측의 이미지 아코디언은 3D 인터랙티브 텍스트 클라우드로 변경하여 파트너십의 규모감을 입체적으로 전달하며, 좌측 텍스트 영역은 "Trusted By" 무한 롤링 텍스트를 과감히 삭제하고 타이포그래피와 여백을 극대화하여 훨씬 더 프리미엄하고 신뢰감 있는 브랜드 이미지를 구축합니다.

---

## 2. 좌측 영역 (Text & Copy) 리디자인 계획

기존 정보가 중복되고 시선이 분산되던 하단 롤링 배너를 삭제함으로써, 시선이 오롯이 헤드라인과 CTA(행동 유도 버튼)에 집중되도록 구조를 개편합니다.

### 2.1 카피라이팅(Copywriting) 고도화
* **기존**: Our Works / 성공의 증명 / 실제 데이터와 성과로 증명된 담하의 포트폴리오. 단순한 디자인을 넘어 브랜딩과 마케팅의 본질을 꿰뚫는 압도적인 결과물을 확인해 보세요.
* **변경안 제안**:
  * **서브 타이틀(Kicker)**: `OUR PARTNERS & WORKS` (자간을 넓게 두어 고급스러움 강조)
  * **메인 타이틀(Headline)**: `숫자와 혁신으로 증명하는` <br/> `압도적 성공의 레퍼런스`
  * **본문(Body)**: `담하는 단순한 에이전시를 넘어, 데이터와 성과로 성공을 증명하는 전략적 파트너입니다. 50+ 이상의 종합병원과 브랜드가 경험한 실질적인 성장, 그리고 담하를 거쳐간 수많은 클라이언트들이 그 결과를 직접 증명하고 있습니다.`
  * _(진행 시 위 문구로 적용하되, 세부 수정 피드백이 있으시면 반영하겠습니다.)_

### 2.2 UI/UX 디자인 디테일 (시니어 디자이너 시각)
1. **타이포그래피 위계(Hierarchy) 및 스타일링**:
   - 메인 타이틀에 담하의 브랜드 컬러(`#D60000`)를 활용한 은은한 그라데이션 포인트 또는 텍스트 하이라이트를 주어 강렬한 첫인상을 남깁니다.
   - 본문 텍스트는 `text-text-sub` 컬러를 유지하되, `leading-relaxed`를 통해 가독성을 높이고 줄바꿈 기호를 최적화합니다.
2. **등장 애니메이션 (Framer Motion)**:
   - 텍스트가 단순히 나타나는 것을 넘어, 각 라인이 `stagger` 효과로 부드럽게(Fade-up) 등장하도록 연출하여 우아함을 더합니다.
3. **CTA 버튼 (Call To Action) 강화**:
   - 버튼에 글래스모피즘(Glassmorphism) 느낌의 반투명 효과나, 호버 시 은은한 레드 글로우(Glow) 효과를 추가하여 고급스러운 인터랙션을 유도합니다.

---

## 3. 우측 영역 (3D 텍스트 기반 Icon Cloud) 구성 계획

기존 IT 로고 위주의 `react-icon-cloud`를 담하의 실제 클라이언트 명단이 떠다니는 3D 공간으로 커스텀 개발합니다.

1. **데이터 소스 매핑**: 
   - `src/data/clients.ts`의 `FEATURED_CLIENTS` 데이터를 직접 활용하여, "일신기독병원", "센텀종합병원" 등의 텍스트를 렌더링합니다.
   - 각 노드 렌더링 시 텍스트 좌/우에 간단한 아이콘(병원 마크 등)이나 담하레드(`bg-primary`) 포인트 닷(Dot)을 배치하여 심심하지 않게 구성합니다.
2. **3D 클라우드 공간(Cloud Environment) 및 물리 효과 설정**:
   - 회전 속도: `maxSpeed: 0.04`, `minSpeed: 0.02`로 느리고 부드럽게 유영하도록 설정.
   - 깊이 및 스케일: `depth: 1`, `imageScale: 2` (또는 텍스트 크기에 맞춰 동적 스케일링 적용)
   - 테마 연동: 라이트/다크 모드에 맞춰 텍스트 컬러 스위칭 보장.
3. **인터랙션 피드백**:
   - 사용자의 마우스 휠이나 드래그에 반응하여 클라우드가 회전하도록 제어하며, 특정 클라이언트 클릭 또는 호버 시 포커싱 효과(다른 텍스트는 옅어짐)를 검토/적용합니다.

## 4. 컴포넌트 통합 및 기술적 단계 (Implementation Steps)

1. **패키지 설치**: `next-themes`, `react-icon-cloud`
2. **신규 컴포넌트 생성**: `src/components/ui/client-icon-cloud.tsx` 생성 및 텍스트 렌더링 커스텀 클래스 구현.
3. **홈 컴포넌트 반영**: 
   - `src/components/home/PortfolioPreview.tsx`의 "Trusted By" 롤링 컨테이너 삭제.
   - 좌측 텍스트 내용물 및 애니메이션 프레임 업데이트.
   - 우측 아코디언 컴포넌트를 `<ClientIconCloud clients={FEATURED_CLIENTS} />`로 통째로 교체.
4. **리뷰 및 QA**: `npm run lint` 및 로컬 서버를 통한 브라우저 렌더링(클라우드 회전 부하 및 텍스트 가독성) 상태 점검.
