# 조직도 업데이트 구현 계획서

## 1. 개요
제공해주신 새로운 조직도 이미지를 바탕으로 `About` 페이지(`/src/app/about/page.tsx`)의 조직도(Organisation Section)를 갱신하는 계획입니다.

## 2. 변경 사항 분석 (기존 vs 신규)
*   **대표이사, 경영기획실, 연구전담:** 기존과 동일하게 유지됩니다.
*   **하위 부서/팀 개편:**
    *   **기존 (3개):** 마케팅부, 웹제작부, 브랜드전략부
    *   **신규 (4개):** 마케팅부, 웹제작부, 미디어컨텐츠기획팀, 기획관리팀
*   **레이아웃 변경:** 하위 부서 영역이 기존 3단 배열(`grid-cols-3`)에서 4단 배열(`grid-cols-4` 또는 화면 크기에 맞춰 `grid-cols-2 lg:grid-cols-4`)로 변경되어야 합니다.

## 3. 상세 수정 계획

**`src/app/about/page.tsx` 파일 내 `ORG_DEPARTMENTS` 배열 데이터 수정**

1.  **마케팅부, 웹제작부:** 기존 데이터 유지
2.  **미디어컨텐츠기획팀 (신설/변경):**
    *   기존 '브랜드전략부'를 대체하는 성격으로 보아 테마 색상을 유지하거나 새로운 색상 부여
    *   `en`: "Media Content" (임의 지정)
    *   `color`: "#8B5CF6" (기존 퍼플 유지)
    *   `services`: 영상 기획, 콘텐츠 기획 디자인 등의 서비스 항목으로 재구성
3.  **기획관리팀 (신설):**
    *   조직의 기획 및 관리를 담당하는 부서
    *   `en`: "Planning & Admin" (임의 지정)
    *   `color`: "#10B981" (신뢰감을 주는 에메랄드 그린 색상 임의 지정)
    *   `services`: 경영 기획, 인사/총무, 재무/회계 등 관리 위주의 서비스 항목으로 구성
4.  *(참고)* 이미지에 자회사(담하손해사정, 담하소개팅) 표기가 생략되어 있습니다. 기존 코드의 `ORG_AFFILIATES` 표기 유지 여부는 원하시는 대로 결정할 수 있습니다. (기본적으로는 유지하는 방향으로 작업하겠습니다.)

**UI 레이아웃 업데이트**
*   부서 카드 영역 CSS 클래스 수정 (3열 구조를 4열 구조 지원 형태로 변경)
    *   기존: `<div className="grid grid-cols-3 gap-6 pt-8">`
    *   변경: `<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-6 pt-8">`

---
위 계획된 내용의 임의 지정 데이터(영문명, 색상, 세부 서비스 항목)에 대해 수정하고 싶으신 부분이 있으시거나, 기존의 '자회사(담하손해사정 등)' 표기를 삭제해야 한다면 말씀해주세요.
