# Subagent Quality Gate Checklist v1

완료 전 필수 체크 항목이다. 4개 영역 모두 충족되어야 완료 처리한다.

## 1. 요구사항 충족

- [ ] 사용자 요청의 핵심 목표를 달성했다
- [ ] 범위 외 수정이 없다
- [ ] 제약사항(DESIGN.md/기술/권한)을 준수했다

## 2. 회귀 위험 점검

- [ ] 기존 동작을 깨뜨릴 수 있는 변경을 점검했다
- [ ] 필요한 검증(빌드/테스트/수동 확인)을 수행했다
- [ ] 검증 불가 항목은 사유를 기록했다

## 3. 문서 및 결정 로그

- [ ] 주요 결정 이유를 기록했다
- [ ] 새 규칙/정책 변경 시 관련 문서를 동기화했다
- [ ] 다음 액션이 명확히 남아 있다

## 4. 검증 증거

- [ ] 변경 파일 목록을 제시했다
- [ ] 실행 결과 또는 확인 증거를 첨부했다
- [ ] 잔여 리스크를 명시했다

## 5. KPI 기록 (주간 회고용)

- [ ] 1회 성공 여부
- [ ] 재작업 발생 여부
- [ ] 완료 시간(상대 비교용)
- [ ] 사용자 재요청 여부
