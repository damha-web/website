## 1. 작업 목적

`/web` 상담 신청 폼 제출 시 카카오워크 채널로 실시간 알림을 전송하고, 연락처 입력 UX를 개선하는 것을 목표로 작업함.

---

## 2. 변경 파일

- `src/app/api/web-contact/route.ts`
- `src/components/web/ContactForm.tsx`

---

## 3. 구현 내용

### A. 카카오워크 웹훅 연동 (`route.ts`)

- 환경변수 `KAKAOWORK_WEBHOOK_URL` 기반 알림 전송
- Webhook 요청 타임아웃(3초) 적용
- Webhook 응답 `ok` 여부 검증 및 실패 로그 기록
- Webhook 실패 시에도 사용자 폼 제출은 성공 처리(알림 실패 격리)
- 로그 내 전화번호 마스킹 처리
- 문의 내용 최대 길이(1000자) 제한

### B. 연락처 입력 UX/검증 (`ContactForm.tsx`)

- 연락처 입력 시 숫자 기반 자동 하이픈 포맷 적용
  - 예: `01012345678` 입력 시 `010-1234-5678`
- 연락처 형식 실시간 검증(blur 이후 안내)
- 잘못된 형식 입력 시 필드 하단 오류 문구 노출
- 제출 시 형식 불일치하면 API 요청 전 차단
