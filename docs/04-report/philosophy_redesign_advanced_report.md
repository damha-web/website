## 2. 주요 구현 내역 (시각적/기술적 측면)

### A. 좌측 켄 번즈(Ken Burns) 이미지 & 글래스모피즘
- **이미지 교체 및 효과**: 기존 코딩 이미지를 요청하신 `pc001007479.jpg`로 전면 교체. `motion.img`를 활용해 이미지가 20초 주기로 미세하게 확대/축소되는 켄 번즈 효과를 적용하여 정적인 화면에 생명력 부여.
- **아날로그 노이즈 텍스처(Grain Effect)**: SVG 기반의 난수형 노이즈 레이어(`mix-blend-overlay`)를 삽입해 렌더링된 화면이 지나치게 디지털스럽지 않고, 매거진이나 포스터처럼 프리미엄한 질감이 나도록 처리.
- **다중 그라데이션 및 명암**: 텍스트의 가독성을 높이면서도 깊이감이 느껴지도록 이중 그라데이션 오버레이(좌측→우측, 하단→상단) 적용.

### B. 압도적인 타이포그래피 (Stagger Animation & Typography)
- **Stagger Reveal**: 메인 제목인 "Why Damha Works"를 알파벳 단위로 쪼개어, 스크롤 진입 시 순차적으로 튀어오르는 스프링 스태거(Spring Stagger) 모션 적용.
- **글로우(Glow) 이펙트**: 핵심인 "결정적 차이" 텍스트에 Damha Red 컬러의 Glow 이펙트(`drop-shadow-[...rgba(214,0,0,0.5)]`)를 줘서 무게감과 흡입력을 강조.
- **프리미엄 인용구 카드**: 텍스트 웨이트의 굵기 대비(font-light, base, extrabold)를 통해 문구의 뉘앙스 차이를 시각화("전술" vs "**전략**"). 블러 효과 위에 `border-white/10`을 조합해 섬세한 글래스모피즘 완성.

### C. 아코디언 마이크로 인터랙션 (Micro-interactions)
- **동적 상태 변화**: 아코디언 항목이 활성화(`isActive`) 될 때, 좌측에 Damha Red 컬러의 세로 라인이 위에서 아래로 스와이프 되듯(`height` 애니메이션) 생성.
- **본문 등장 모션**: 단순히 펼쳐지는 것이 아니라, 내용이 약간의 투명도와 블러(Blur) 상태에서 선명해지며 위로 떠오르는 Fade-up 효과 적용.
- **Hover/Active 컬러링 트랜지션**: 우측 화살표/플러스 아이콘이 마우스 호버 시와 활성화 시에 자연스럽게 배경과 테두리 컬러가 교차되며 45도 부드럽게 회전하도록 구현.
- **스탯(Stats) 뱃지 강조**: 설명 텍스트 내부의 숫자/통계 요소에 `animate-pulse`가 적용된 레드 닷(Red dot)을 추가해, 데이터 중심의 신뢰감 형성.
