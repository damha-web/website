### 카테고리 B: 서비스/상품 컨텐츠
**적용 위치:** `Products.tsx`, `/services` 페이지

#### B-1. 브랜딩 서비스
```
타이틀: Branding
부제: 병원 아이덴티티의 핵심을 설계합니다
설명: 비주얼 아이덴티티(VI) 개발부터 홈페이지 제작까지,
      브랜드의 첫인상을 전략적으로 설계합니다.

세부 서비스:
- VI (비주얼 아이덴티티): 로고, 컬러, 타이포그래피 등 브랜드 시각 시스템
- 홈페이지 제작: 1page 모달팝업강조형 / 일반형 2종류
  → 1page형: 엄지 하나로 전체 탐색, 모바일 최적화
  → 일반형: 다중 페이지 구조, 진료과목별 심화 정보
- 사진 촬영 (Photography)
- 모션그래픽 (Motion)
```

#### B-2. 마케팅 서비스
```
타이틀: Marketing
부제: 환자가 찾아오는 온라인 환경을 만듭니다
설명: 블로그부터 유튜브까지, 고객 여정 전 단계를
      데이터 기반으로 최적화합니다.

세부 서비스:
- 네이버 블로그: 콘텐츠 제작 + 노출 품질 관리
- 네이버 카페: 댓글 관리, 바이럴 마케팅
- SNS (인스타그램): 릴스 제작, 팔로우 이벤트, 인사이트 분석
- 유튜브: 영상 콘텐츠 기획 및 채널 관리
- 검색광고: 네이버/구글 키워드 광고
- 인플루언서/체험단
- 온라인 언론보도: 전문 칼럼형 보도자료
- 퍼포먼스 광고: 카카오 디스플레이/비즈보드, 당근 배너
```

#### B-3. 디자인 서비스
```
타이틀: Design
부제: 병원 밖에서도 브랜드가 살아 숨 쉽니다
설명: 오프라인 접점에서 브랜드를 강화하는
      고품질 디자인 제작 서비스

세부 서비스:
- 오프라인 시안물 디자인
- 인쇄물 디자인 제작
```

#### B-4. 오프라인 광고 (신규 카테고리)
```
타이틀: Offline AD
부제: 지역 환자를 직접 만나는 오프라인 광고
설명: 버스 광고부터 택배차 랩핑까지, 지역 밀착형
      광고로 인지도를 극대화합니다.

세부 서비스:
- 투입카드/전단지 배포 (시장조사 포함)
- 시내버스 광고: 하차문A(800×500), 하차문B(450×600), 시트커버(255×140), LED+음성광고

- 택배차 랩핑: 3면 랩핑
- 버스 음성 광고: 정류장 안내방송 후 12초 독점 노출
```

#### B-5. 컨설팅 (기존 유지 + 강화)

타이틀: Consulting
부제: MOT 기반 병원 브랜딩 전략
설명: 환자 여정 7단계(인지→탐색→예약→방문→진료→사후관리→재방문)를
      분석하여 각 접점에서 최적의 브랜드 경험을 설계합니다.

핵심 서비스:
- MOT 기반 브랜딩 설계
- 광고 고객 여정 컨설팅 (온라인-오프라인 통합)
- 월간 마케팅 보고 및 전략 수립
- 불만족 리뷰 대응 (위임장 기반 법적 대응 포함)
