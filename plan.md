## 변경 사항

### 수정 파일: `src/components/home/PhilosophySection.tsx`

#### 레이아웃 변경
- **기존**: `lg:w-[35%]` sticky 이미지 + `lg:w-[65%]` 스크롤 콘텐츠 (space-y-[60vh])
- **변경**: `lg:w-1/2` 이미지 (섹션 높이 꽉 참) + `lg:w-1/2` 아코디언 리스트
- 섹션 높이: 뷰포트 한 화면 (`min-h-screen`) — 긴 스크롤 제거
- sticky/parallax 효과 제거 (간결한 한 화면 구성)

#### 좌측 이미지 영역
- 기존 `aspect-[4/5]` 둥근 카드 → 섹션 높이 전체를 채우는 이미지
- `object-cover` + 그라데이션 오버레이
- 이미지 위에 인용구 텍스트 배치 (beautifulmom 스타일)
- 사용 이미지: `/assets/images/code.jpg` (기존 유지)

#### 우측 아코디언 영역
- 상단: 섹션 태그 + 제목 ("Why Damha Works" + "담하가 만드는 결정적 차이")
- 3개 아코디언 항목:
  - 번호 (01, 02, 03) + 제목 (볼드)
  - 호버/클릭 시 설명 텍스트 슬라이드 다운
  - 항목 사이 구분선 (border-b)
  - 첫 번째 항목 기본 열림

#### 애니메이션
- 아코디언 열기/닫기: framer-motion `AnimatePresence` + height 애니메이션
- 호버 인터랙션: 데스크톱에서 호버 시 열림, 모바일에서 클릭 시 열림
- 섹션 등장: whileInView fadeIn (간단하게)

#### 모바일 반응형
- `lg` 이하: 이미지 상단 + 아코디언 하단 세로 스택
- 이미지 높이: `h-[40vh]` 또는 `h-[300px]`
- 아코디언: 클릭 기반 토글 (호버 비활성)

### 제거 항목
- `useScroll`, `useTransform` (parallax 효과)
- `TextReveal` 컴포넌트 (간결한 텍스트로 대체)
- `customFadeInUp` (간단한 fadeIn으로 대체)
- `damha_mesh_decor.png` 배경 (좌측 이미지로 대체)
- `space-y-[60vh]` 긴 스크롤 구조

### 유지 항목
- `PHILOSOPHY_CONTENT` 데이터 (3개 항목)
- framer-motion (AnimatePresence로 용도 변경)
- 디자인 토큰 (primary, secondary 컬러)
