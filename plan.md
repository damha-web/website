# 웹제작부 서브페이지 — 와이어프레임 설계 계획

## 개요

**라우트**: `/web` (원페이지 스크롤 랜딩)
**목적**: 영업 + 브랜딩 + 리드 생성 종합 포트폴리오
**타겟**: 병원/의료기관 (1인 병원 ~ 대형 병원)
**구조**: 10개 섹션 순차 스크롤 + 포트폴리오 상세는 모달 or 기존 `/portfolio/[slug]` 연계

## 섹션별 인터랙션 상세

| 섹션 | 인터랙션 | 구현 방식 |
|------|----------|----------|
| Hero | 배경 디바이스 목업 슬라이드 | Swiper 또는 framer-motion AnimatePresence |
| Problem | 카드 순차 등장 | framer-motion staggerChildren |
| About | 레이더 차트 애니메이션 | SVG + framer-motion path drawing |
| Difference | 호버 시 카드 뒤집기 | CSS 3D transform (perspective + rotateY) |
| Service | 상품 카드 호버 강조 | scale + shadow transition |
| CMS | 탭 전환 슬라이드 | framer-motion layout animation |
| Portfolio | 필터 전환 + 모달 | framer-motion AnimatePresence + Dialog |
| Process | 스크롤 시 단계 활성화 | IntersectionObserver + 프로그레스 바 |
| Contact | 폼 제출 피드백 | 로딩 상태 + 성공/에러 토스트 |
